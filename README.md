## 📖 About This Project

This is my **personal portfolio website** showcasing my journey as a Frontend Team Lead and Engineer. Built with pure HTML, CSS, and JavaScript — no frameworks — demonstrating my fundamental web development skills.

## ✨ Features

### 🖱️ Interactive Elements

- **Particle Background** — Dynamic canvas-based particle system
- **Typewriter Effect** — Dynamic text animation in hero section
- **Scroll Reveal** — Elements fade in as you scroll

### 🎨 Visual Design

- **Dark Theme** — Modern dark mode design
- **Glassmorphism** — Frosted glass UI effects
- **Gradient Accents** — Vibrant blue-to-pink gradients
- **Responsive Layout** — Mobile-first, fully responsive

### ⚡ Performance Features

- **Lazy Loading** — Images and sections load on demand
- **Optimized Animations** — CSS transforms instead of layout changes
- **Accessible** — Semantic HTML, ARIA labels
